# Lernziele Niveau g – Online App

Diese Mini-App ist eine statische Website (HTML/CSS/JS) und kann gratis online gehostet werden.

## Option A: GitHub Pages (gratis)
1. Erstelle ein GitHub-Konto (falls nötig).
2. Neues Repository erstellen, z.B. `lernziele-niveau-g`.
3. Lade **alle Dateien aus diesem Ordner** hoch (index.html, sw.js, manifest..., icons/).
4. Im Repository: **Settings → Pages**
5. Source: **Deploy from a branch** → Branch: `main` → Folder: `/ (root)` → Save
6. Nach 1–2 Minuten ist deine Seite online unter:
   `https://DEINNAME.github.io/lernziele-niveau-g/`

## Option B: Netlify (Drag & Drop, gratis)
1. Gehe zu Netlify und logge dich ein.
2. Ziehe den ganzen Ordner (oder die ZIP entpackt) in den Bereich „Deploy“.
3. Du bekommst sofort eine URL (z.B. `https://xyz.netlify.app`).

## PWA (wie eine App)
- Auf iPad/iPhone: Safari → Teilen → „Zum Home-Bildschirm“.
- Offline: Service Worker cached die wichtigsten Dateien.
